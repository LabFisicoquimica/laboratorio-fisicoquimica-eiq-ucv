# **Laboratorio de Fisicoquímica – UCV**

Bienvenido a la página web del Laboratorio de Fisicoquímica de la Escuela de Ingeniería Química de la Universidad Central de Venezuela (UCV).

## Contenidos

- Información sobre el curso
- Programa de prácticas
- Enlaces a guías y videos
- Horarios
- Material adicional

## ¿Cómo usar este sitio?

Simplemente abre el archivo `index.html` en tu navegador para explorar todos los recursos del laboratorio.
